Bridge supports the following fiat currencies and rails.

| **Currency** | **Supported Rails**                                               |
| :----------- | :---------------------------------------------------------------- |
| USD          | Next Day ACH, Same Day ACH (outgoing transactions only), and Wire |
| EURO         | SEPA                                                              |
| MXN          | SPEI                                                              |
